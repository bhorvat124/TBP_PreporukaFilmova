Upute za instalaciju

Python

Pripremljeni programi su namijenjeni izvođenju u Windows okruženju na sustavu koji ima NVIDIA grafičku karticu s CUDA podrškom. Kako bi izvođenje bilo moguće, potrebno je imati sljedeće:

  1. Python i potrebni moduli
  2. CUDA ToolKit i NVCC kompajler
  3. Nvidia grafičku karticu

NVIDIA nudi Python Wheels za instalaciju CUDA-e putem pip-a, prvenstveno za korištenje CUDA-e s Pythonom. Ovi su paketi namijenjeni za korištenje tijekom izvođenja i trenutno ne uključuju razvojne alate (mogu se instalirati zasebno).

py -m pip install --upgrade setuptools pip wheel // nakon ovoga moguće je instalirati nvidia-pyindex modul

py -m pip install nvidia-pyindex

py -m pip install nvidia-cuda-runtime-cu11

Moduli

pip install opencv-python // Za instalaciju OpenCV, kako bi se omogućio prikaz slike i primjena mape boja

pip install numbaZa // Python kompajler koji omogućuje i korištenje CUDA-e

pip install numpy // Za instalaciju numpy, za matematičke izračune

Uz pomoć PowerShell-a se je potrebno se je pozicionirati unutar direktorija u kojem se nalaze .py datoteke te naredbom python .\gpu_fractal.py pokrenuti program

C++

Za C++ verziju koda potrebno je imate sljedeće:

  1. Visual Studio 2019 ili noviji
  2. CUDA ToolKit i NVCC kompajler
  3. Nvidia grafička kartica
  4. CMake

OpenCV i CMake

Potrebno je preuzeti OpenCV ovdje i instalirati.

Potrebno je preuzeti CMake ovdje i instalirati.

Nakon instalacije potrebno je namjestiti Enviroment variables. Nalaze se u Control Panelu. Na kartici Advanced, pod System variables nalazi se Path varijabla. Ovdje je potrebno specificirati putanju do bin i lib direktorija koji se generiraju prilikom instalacije Open CV-a. Isto tako potrebno je specificirati i dodatni putanju do CMake bin direktorija.

Nakon toga potrebno je otvoriti PowerShell i napraviti sljedeće:
Uz pomoć cd naredbe se je potrebno pozicionirati na direktorij u c++ direktorij (direktorij u kojem se nalazi datoteka CMakeLists.txt i datoteke izvornog koda te direktorij include).
Nakon toga je potrebno kreirati novi direktorij naziva build: mkdir build

Za kraj je potrebno kompajlirati datoteke i kreirati izvršnu datoteku:

cmake.exe -S . -B .\build\
cmake.exe --build .\build\ --config Debug --target ALL_BUILD -j 14 --

Kako bi se pokrenuo kompajlirani program, potrebno je otići u direktorij build/Debug/ u kojem je moguće pronaći datoteku mandelbrotCUDA.exe, čijim se pokretanjem pokreće program
